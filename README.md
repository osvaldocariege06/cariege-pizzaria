# Pizzaria Cariege

Este é um aplicativo de pizzaria desenvolvido para a plataforma Android utilizando Kotlin e Jetpack Compose.

## Como Usar

1. Após a instalação, abra o aplicativo.
2. Faça login ou crie uma nova conta.
3. Navegue pelo cardápio, escolha suas pizzas e adicione ao carrinho.
4. Finalize o pedido e acompanhe o status.

## Ferramentas e Tecnologias

*   **Linguagem:** Kotlin
*   **Interface de Usuário:** Jetpack Compose
*   **IDE:** Android Studio
*   **Sistema de Build:** Gradle

## Como Compilar e Executar o Projeto

1.  Clone este repositório para a sua máquina local.
2.  Abra o projeto com o Android Studio.
3.  Aguarde o Gradle sincronizar as dependências.
4.  Execute o aplicativo em um emulador Android ou em um dispositivo físico.
